# Why-Do-Chameleons-Change-Colors
WORK IN PROGRESS

'Why Do Chameleons Change Colors?' is an explorable explanation that aims to simplify the complex mechanisms that allow the color-changing abilities of chameleons. This website was originally an assignment in Advanced Interaction Design (GDES5341). However, I have been expanding on this project to add more visuals and interactive animations using the p5.js JavaScript library.
